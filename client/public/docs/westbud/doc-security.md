# Политика Безопасности

### 1. Защита от Спама
*   **Honeypot:** В форме используется скрытое поле `company_role`. Если бот его заполнит — заявка молча отклоняется.
*   **Rate Limiting:** Ограничение на отправку форм с одного IP (например, не чаще 1 раза в минуту), реализуется на бэкенде или через сервис форм (EmailJS/Formspree).

### 2. HTTPS и SSL
*   Сайт должен быть доступен **только** по протоколу HTTPS.
*   Настроен редирект с HTTP на HTTPS.

### 3. Защита Контента
*   Запрет на вставку сайта в `iframe` на чужих ресурсах (заголовок `X-Frame-Options: DENY`) для защиты от Clickjacking.
