# Руководство для Разработчика

### 1. Правило «Чистого Скальпеля»
При удалении Auth-кода всегда запускай `npm run build` после каждого удаленного файла. Это позволит мгновенно найти места, где нарушились зависимости.

### 2. Работа с формами
Если нужно добавить новое поле в форму:
1. Обнови `LeadSchema` в `src/lib/validations.ts`.
2. Добавь `Input` в компонент `ContactForm.tsx`.
3. Обнови шаблон сообщения в `api/send-lead.ts`.

### 3. Тестирование
Перед деплоем проверь:
- [ ] Форма не отправляется, если поля пустые.
- [ ] Telegram получает сообщение даже при очень длинном описании (обрезка текста).
